## 📖 模块安装教程
### 1. Magisk 模块安装
1.  下载对应模块的 `.zip` 安装包
2.  打开 Magisk App → 点击「模块」→ 「从本地安装」
3.  选择下载的 `.zip` 文件，等待刷入完成后**重启设备**
4.  若出现卡开机/报错，请在Recovery中卸载对应模块

### 2. 刷机包安装
1.  下载对应刷机包，放入手机内置存储
2.  进入第三方Recovery（如OrangeFox）
3.  选择「安装」→ 找到刷机包 → 滑动刷入 → 重启设备

## 🚀 新手必看：模块安装顺序
1.  解锁Bootloader → 刷入第三方Recovery → 刷入Magisk获取Root
2.  开启Magisk的Zygisk功能 → 刷入Zygisk Next增强稳定性
3.  刷入SafetyNet Fix解决银行APP闪退问题
4.  依次刷入音效、美化、性能、功能类模块
5.  **每刷1-2个模块就重启一次**，避免一次性刷太多导致冲突

## 🆘 常见问题排查
| 问题 | 解决方法 |
|------|----------|
| 刷完模块卡开机 | 进入Recovery，卸载最后刷入的模块 |
| 音效模块没效果 | 检查是否与其他音效模块冲突，重启后再试 |
| 电池容量显示不准 | 刷入电池解容模块，校准电池信息 |
| 游戏帧率不稳 | 刷入温控优化模块，关闭后台多余应用 |
| 指纹/相机失效 | 卸载最近刷入的模块，恢复系统完整性 |
